Wit.ai

The Wit API makes it easy for developers to add Siri-like voice interfaces and natural language processing to their apps. The API accepts voice and text commands and turns them into actionable data. There are no complicated rules or grammars to set up. Simply provide Wit with a few examples in the convenient web console and you’re ready to go. You can get started in 5 minutes.

We provide iOS, Android and Web SDKs to help you stream audio to the to the Wit API. After you send your voice or text command to Wit, we'll figure out the user's intent and send back JSON. So if your user says, “Wake me up tomorrow at 7,” Wit will send back something like {“intent”: “set_alarm”, “datetime”: “20140920T070000”}.
